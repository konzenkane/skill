## 🚀 快速开始

### GitHub Pages 部署（推荐）

```bash
# 1. 解压
tar -xzf skill-store-website-FIXED.tar.gz

# 2. 上传到 GitHub
git init
git add .
git commit -m "🦞 AI龙虾技能严选 v1.0"
git remote add origin https://github.com/你的用户名/skill-store.git
git push -u origin main

# 3. 开启 Pages
# Settings → Pages → Source: main branch
```

### 访问网站

- 前端: `https://你的用户名.github.io/skill-store/client/`
- 后端: 需要部署到 Vercel/Render
